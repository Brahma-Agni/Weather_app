# **App Name**: Weather Parader

## Core Features:

- Location Input: Allow users to input a location (current, custom, or journey waypoints).
- Weather Prediction: Generate a weather prediction based on the userâ€™s inputted location.
- Probabilistic Analysis: Employ a tool, powered by generative AI, that will express weather forecasts not as certainties, but as probabilities that can affect users' plans and decisions.
- Hourly and Daily Forecasts: Show hourly and daily weather forecasts (temperature, precipitation, wind, etc.).
- Interactive Map: Render a map showing the location and the associated weather conditions. Visualize weather patterns, such as approaching rain, or other phenomena. Show a route with weather pins (journey).
- Weather Risk Score: Use generative AI to calculate and show a simple weather risk score or badge for each time point in journey, along with an explanation of why that score was given.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4), symbolizing clarity and trustworthiness.
- Background color: Light gray (#F5F5F5), providing a neutral and clean backdrop.
- Accent color: Warm orange (#FF8A65), used to highlight important data and calls to action.
- Body font: 'PT Sans', sans-serif, for readability and modern look.
- Headline font: 'Playfair', serif, for a touch of elegance.
- Use outlined icons from 'Lucide-react or Heroicons', simple and easily understandable.
- Subtle transitions for map interactions and loading states.
